## 🎯 Recommended Learning Path

### Beginner Level
1. Start with **C/fundamentals** for basic programming concepts
2. Try **C/algorithms** for simple algorithm implementations
3. Explore **Python/algorithms** for more readable examples

### Intermediate Level
1. Study **data_structures** across all languages
2. Practice with **mathematical** examples
3. Build **games** to apply concepts

### Advanced Level
1. Compare implementations across languages
2. Optimize algorithms for performance
3. Create your own examples
